创建git项目相关命令

2017-04-17

今天学习git

安装gitbash



git使用前设置：$ git config –global user.name “YOUR NAME”

$ git config –global user.email “Your Email”


查看git设置情况：$ git config -l


创建git项目：建立新文件夹，然后当前文件夹下 $ git init


命令：l -a 查看隐藏文件，轻易不要对该隐藏文件进行操作修改


命令：git n 查看git仓库状态


命令：git add index.html 将文件提交到暂存区


命令：git commit -m “我的第一次提交代码” 将文件保存到仓库


命令：git checkout – index.html 放弃修改的代码


命令：git log 查看日志


命令：git reset 日志显示的版本名 –hard 回到某一次的文件版本


命令：git add -A 将多个文件提交到暂存区


命令：git commit -m “我这次提交多个文件” 将多个文件保存到仓库


ssh公钥：本地仓库和在线仓库数据传输是通过ssh加密的。所以需要一个加密钥匙来确定身份


第一步：创建公钥—-ssh-keygen -t rsa -C “961522375@qq.com”


第二步：查看公钥:在c盘用户文件夹下，你的用户名文件下有个.ssh文件夹，打开，公钥就在里面


在github创建新项目，在该项目下setting里填写ssh公钥，与本地仓库建立连接


在github新项目下找到clone and download按钮，点击按钮，复制里面的地址



在source tree里点击克隆新建，克隆仓库，在源url里粘贴地址，目标路径为克隆的本地仓库路径


然后就可以通过source tree对仓库进行数据提交修改
